# Mineradio - LX Music（落雪音乐）/ MR

> **项目来源与署名：** 本仓库基于原作者 **[@XxHuberrr](https://github.com/XxHuberrr)** 的开源项目 **[XxHuberrr/Mineradio](https://github.com/XxHuberrr/Mineradio)** 进行非官方二次开发。感谢原作者及原项目所有贡献者。本仓库保留原项目版权与 MIT License 声明，不主张替代原作者身份，也不代表原作者为本版本提供背书、维护、授权或担保。

MR 是一个以**本地音乐播放、歌词展示和音乐可视化**为核心的桌面应用。用户可以扫描并播放自己有权使用的本地音频文件，不需要安装 LX Music，也不需要登录任何第三方音乐平台账号。

> **重要说明**
>
> - 本仓库不预置、不托管、不销售、不推荐或分发第三方音乐音源、音源脚本、账号凭据、Cookie、Token、会员权益或受版权保护的音频文件。
> - 项目不提供 QQ 音乐、网易云音乐或其他音乐平台的账号登录功能。
> - 用户不得使用本项目绕过付费限制、数字版权管理、访问控制、地区限制、会员验证或其他技术保护措施。
> - 涉及第三方平台的数据搜索、歌单导入、歌词、封面或在线播放时，用户应确保所使用的接口、数据源和内容已获得合法授权，并遵守相关平台服务条款及当地法律法规。

[下载 Windows 安装包](https://github.com/ww085213/Mineradio-LX-Music/releases/download/v1.5.1/Mineradio.Setup.1.5.1.exe) · [查看发布说明](RELEASE_NOTES_1.5.1.md)

> v1.5.1 可直接覆盖安装旧版本。用户设置、自建歌单、用户自行添加的数据源配置和本地音乐索引通常会继续保留，升级前仍建议自行备份重要数据。

## 1.5.1 更新内容

- **播放器内歌词：** 支持左侧、居中、右侧、拖拽和重置，位置可保存。
- **桌面歌词：** 优化隐藏/最小化后的流畅度，增强清晰度，修复拖动、滚轮调大小和抢焦点问题。
- **音源管理：** 支持删除音源，音源较多时列表可滚动。
- **本地音乐：** 支持 MP3、FLAC、WAV、OGG、M4A，增强封面、歌词、WAV/M4A 标签读取。
- **歌单导入：** 优化超过 120 首歌单的分页导入，在线歌单详情支持加载更多。
- **左侧歌单：** 恢复当前、本地、加入歌单搜索，首页歌单可定位到左侧。
- **壁纸：** 增加壁纸镜像开关。
- **安装流程：** 第一个“下一步”选择安装位置，第二个“下一步”直接开始安装。

## 支持的内容来源

### 本地内容

- MP3 / FLAC / WAV / M4A / OGG
- 本地文件夹音乐库
- 本地自建歌单
- 本地歌词与封面

### 可选扩展

- LX Music `.lxmc` 歌单文件导入
- 用户主动开启的 LX Music 本机开放 API 联动
- 用户自行配置的兼容数据源
- 公开分享页面或歌单元数据导入
- 用户自行配置的歌词与翻译歌词补查

> 可选扩展不代表相关第三方平台对本项目进行了授权、合作或背书。

## 主要功能

- 本地音乐播放、本地音乐库和自建歌单管理
- 歌词舞台、桌面歌词和歌词位置调节
- 本地封面、本地歌词、内嵌标签读取
- 3D 歌单架和音乐可视化
- 本地图片/视频背景与 Wallpaper Engine 背景读取
- 可选的 LX Music 本机联动与歌单元数据导入
- 用户自行配置的数据源兼容框架

## 不包含什么

MR 不包含或不提供以下内容：

- LX Music 程序本体
- 任何音乐平台账号登录
- 官方平台会员权益
- 第三方账号 Cookie、Token、密钥或访问凭据
- 预置的第三方音乐音源或音源脚本
- 歌曲、专辑或其他受版权保护的音频文件
- 用于破解加密音乐格式的功能
- 绕过付费、会员、地区、数字版权管理或访问控制的功能
- 对第三方平台接口持续可用性的保证
- 对用户自行配置数据源的授权或合法性担保
